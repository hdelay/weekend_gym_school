학교밖 프로그램 신청

학교밖모집일정표 해당 일정표는 시도/시군구에서 변경 요청이 있을 경우 모집 일정이 변경 될 수도 있으니 정확한 일정은 꼭 해당 시군구에 확인 하시기 바랍니다.
※ 해당 일정표는 프로그램 모집 시작일 및 시작 시간을 의미하며 마감일자는 프로그램 모집 상세일정을 확인 하시기 바랍니다.
